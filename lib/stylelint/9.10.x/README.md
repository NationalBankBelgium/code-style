## About

This sub-package hosts the [stylelint](https://stylelint.io) configuration for stylelint 9.10.x.

## Usage

Adapt the content of your `.stylelintrc` file as follows:

```json
{
	"extends": "@nationalbankbelgium/code-style/stylelint/9.10.x",
	"rules": {
		// your rules
	}
}
```
