## About

This sub-package hosts the [stylelint](https://stylelint.io) configuration for stylelint 13.0.x.

## Usage

Adapt the content of your `.stylelintrc` file as follows:

```text
{
	"extends": "@nationalbankbelgium/code-style/stylelint/13.0.x",
	"rules": {
		// your rules
	}
}
```
