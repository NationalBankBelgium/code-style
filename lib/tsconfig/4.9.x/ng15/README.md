## About

This sub-package hosts the [TypeScript](https://www.typescriptlang.org/) configuration for TypeScript 4.9.x. and Angular 15.1.x || 15.2.x

## Usage

Adapt the content of your `tsconfig.json` file as follows:

```text
{
	"extends": "@nationalbankbelgium/code-style/tsconfig/4.9.x/ng15",
	"compilerOptions": {
		// your configuration
	}
}
```
